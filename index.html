👤 Gestão de clientes
/gerar [qtd]
Gera tokens de 30 dias. Ex: /gerar 3
/clientes
Lista todos os clientes com status e validade.
/renovar CHAT_ID
Renova acesso por mais 30 dias.
/revogar CHAT_ID
Revoga acesso imediatamente.
/stats
Resumo: ativos, expirados, tokens livres, receita.
⚙️ Configurar cliente
/setmodo CHAT_ID completo|basico
Define o modo de acesso do cliente.
/setsite CHAT_ID URL
Define site personalizado no rodapé.
/setsite CHAT_ID remover
Remove o site personalizado.
