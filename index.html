Referência de comandos
⚙️ Admin — Tokens
/gerar [qtd]
Gera 1 ou mais tokens de 30 dias. Ex: /gerar 3
⚙️ Admin — Clientes
/clientes
Lista todos os clientes com status, validade e modo.
/renovar CHAT_ID
Renova +30 dias a partir da validade atual.
/revogar CHAT_ID
Bloqueia o acesso imediatamente.
/stats
Resumo: ativos, expirados, tokens livres, receita.
⚙️ Admin — Configurar cliente
/config CHAT_ID modo completo
Posts com botão "Visite nosso Site". Padrão.
/config CHAT_ID modo simples
Posts só com botão "Assistir Agora". Sem link de site.
/config CHAT_ID site https://...
Define site personalizado no rodapé do cliente.
/config CHAT_ID site remover
Remove site personalizado, volta ao padrão.
👤 Cliente — Ativação
/start
Exibe boas-vindas e instruções de como configurar o bot.
/ativar TOKEN
Ativa o plano no canal. Ex: /ativar SF-ABC123XYZ
/meuplano
Mostra status e dias restantes do plano.
